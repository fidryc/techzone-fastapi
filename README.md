# 🛒 TechZone E-Commerce API

TechZone-fastapi — это E-Commerce api с использованием современных технологий и актуальной архитектурой. Api предоставляет полный набор функций для управления товарами, заказами, пользователями и поиском.

## 🚀 Технологический стек

| Компонент | Технологии |
|-----------|------------|
| **Backend** | Python, FastAPI, Uvicorn |
| **База данных** | PostgreSQL, SQLAlchemy 2.0, Alembic |
| **Кэширование** | Redis, FastAPI-Cache |
| **Очереди задач** | Celery (Redis broker), RabbitMQ |
| **Поиск** | Elasticsearch |
| **Мониторинг** | Prometheus, Grafana |
| **Контейнеризация** | Docker, Docker Compose |
| **Тестирование** | Pytest, pytest-asyncio, pytest-cov |

## 📋 Основные возможности

### 👤 Управление пользователями
- Регистрация/авторизация по email или телефону
- JWT-аутентификация (access + refresh токены, blacklist refresh токенов)
- Разграничение ролей: `user`, `seller`, `admin`
- Подтверждение регистрации через email-код
- Удаление аккаунта с проверкой активных заказов

### 📦 Управление товарами
- CRUD операции (только для seller/admin)
- Фильтрация по цене, рейтингу, категории, характеристикам (JSONB)
- Возможность добавлять новые категории, товары с разными характеристиками и создавать для них валидацию с помощью Pydantic
- Автоматическое обновление среднего рейтинга (Celery-задача)
- Уведомления пользователей о новых товарах в любимых категориях

### 🔍 Поиск
- Полнотекстовый поиск через Elasticsearch
- NGram-анализатор для частичного совпадения (3-15 символов)
- Сохранение истории поисковых запросов
- Кэширование результатов поиска

### 🛍️ Корзина и заказы
- Добавление/удаление товаров из корзины
- Создание заказов с самовывозом или доставкой
- Проверка наличия товаров на складе
- Отслеживание статуса заказа (`new` → `confirmed` → `preparing` → `delivered` → `finished` / `cancelled`)
- Уведомление курьерской службы через RabbitMQ при заказе с доставкой

### 🎯 Рекомендательная система
- Рекомендации на основе любимых категорий пользователя
- Фильтрация по средней цене покупок пользователя
- Рекомендации для новых пользователей (популярные товары)

### 🏬 Магазины и наличие
- Управление магазинами
- Контроль остатков товаров
- Автоматическое списание товаров при оформлении заказа

### 🔧 Административная панель
- SQLAdmin панель для управления всеми сущностями
- Защита middleware-проверкой прав admin
- Просмотр и редактирование: пользователи, заказы, товары, категории, магазины

### 📊 Мониторинг и observability
- Prometheus метрики
- JSON-логирование с кастомным форматтером
- Дашборды Grafana (pre-configured)
- Отслеживание времени выполнения запросов

## ⚙️ Запуск проекта

### Предварительные требования

Перед запуском убедитесь, что установлены:

* Docker
* Docker Compose
* Git

---

### 1. Клонирование проекта

```bash
git clone https://github.com/<your-username>/techzone-fastapi.git
cd techzone-fastapi
```

---

### 2. Создание `.env`

Создайте файл `.env` в корне проекта и заполните необходимые переменные окружения.

Пример:

```env
MODE=DEV
LOG_LEVEL=INFO

DB_HOST=db
DB_PORT=5432
DB_NAME=techzone
DB_USER=postgres
DB_PASS=postgres

REDIS_HOST_=redis
REDIS_PORT_=6379

ELASTIC_HOST=elasticsearch
ELASTIC_PORT=9200

RABBITMQ_HOST=rabbitmq
RABBITMQ_PORT=5672
RABBITMQ_DEFAULT_USER=guest
RABBITMQ_DEFAULT_PASS=guest

INDEX_PRODUCTS=products
```

> Остальные переменные можно заполнить по примеру существующего `.env.example`.

---

### 3. Запуск контейнеров

Собрать и запустить сервисы:

```bash
docker compose up --build
```

Запуск в фоне:

```bash
docker compose up -d --build
```

---

### 4. Создание индекса Elasticsearch

После старта приложения:

```bash
curl -X POST http://127.0.0.1:8000/el/add_all_products_sync
```

или через Swagger UI.

---

### 6. Проверка запуска

После успешного запуска будут доступны:

| Сервис      | URL                          |
| ----------- | ---------------------------- |
| API         | `http://127.0.0.1:8000`      |
| Swagger     | `http://127.0.0.1:8000/docs` |
| Grafana     | `http://127.0.0.1:3000`      |
| Prometheus  | `http://127.0.0.1:9090`      |
| RabbitMQ UI | `http://127.0.0.1:15672`     |

Grafana:

```text
login: admin
password: admin
```

RabbitMQ:

```text
login: guest
password: guest
```

---

### Остановка проекта

```bash
docker compose down
```

Удалить вместе с томами:

```bash
docker compose down -v
```

## 📡 API хендлеры

### 👤 Пользователи `/users`

| Метод | Эндпоинт | Описание |
|---|---|---|
| `POST` | `/users/register_with_email` | Инициализация регистрации по email. Генерирует код подтверждения, сохраняет данные во временное хранилище, выдаёт токен подтверждения в cookie |
| `POST` | `/users/register_with_number` | Инициализация регистрации по номеру телефона |
| `POST` | `/users/verify_code` | Подтверждение кода регистрации. При совпадении кода создаёт пользователя в БД |
| `POST` | `/users/login_with_email` | Вход в аккаунт по email и паролю |
| `POST` | `/users/login_with_number` | Вход в аккаунт по номеру телефона и паролю |
| `POST` | `/users/logout` | Выход из аккаунта, удаление access/refresh токенов из cookie |
| `POST` | `/users/delete_user` | Удаление аккаунта. Отказывает, если у пользователя есть активные заказы |

### 📦 Товары `/products`

| Метод | Эндпоинт | Описание |
|---|---|---|
| `GET` | `/products/search_products/{query_text}` | Полнотекстовый поиск товаров через Elasticsearch (кэш 180 сек) |
| `POST` | `/products/search_products_with_history/{query_text}` | Поиск товаров с сохранением запроса в историю пользователя |
| `GET` | `/products/get_history_queries` | Получение истории поисковых запросов текущего пользователя |
| `GET` | `/products/catalog/{category}/` | Получение товаров категории с фильтрами: цена, рейтинг, гарантия, страна производства, произвольные JSONB-характеристики (кэш 180 сек) |
| `POST` | `/products/add_product` | Добавление нового товара (только для роли `seller`), с валидацией характеристик и опциональной рассылкой уведомлений о новом товаре |
| `GET` | `/products/recomendation` | Персональные рекомендации товаров на основе любимых категорий и средней цены покупок (кэш 30 сек) |
| `GET` | `/products/all` | Получение полного списка всех товаров |
| `GET` | `/products/find_by_id` | Поиск товара по `product_id`, с увеличением счётчика просмотров |
| `GET` | `/products/get_category_spec_schema` | Получение примера схемы характеристик для указанной категории |

### 🛍️ Корзина и заказы `/orders`

| Метод | Эндпоинт | Описание |
|---|---|---|
| `POST` | `/orders/add_to_basket` | Добавление товара в корзину текущего пользователя |
| `POST` | `/orders/create_order_pickup` | Создание заказа с самовывозом: проверка наличия товаров в магазине, списание остатков, очистка корзины |
| `POST` | `/orders/create_order_delivery` | Создание заказа с доставкой: формирование заказа и отправка уведомления курьерской службе через RabbitMQ |
| `GET` | `/orders/get_orders` | Получение списка всех заказов текущего пользователя |
| `PATCH` | `/orders/set_status` | Изменение статуса заказа (доступно только ролям `seller` и `admin`) |

### 🏬 Магазины `/store`

| Метод | Эндпоинт | Описание |
|---|---|---|
| `GET` | `/store/get_quantity_of_product/{store_id}/{product_id}` | Получение остатка конкретного товара в конкретном магазине |

### 🔍 Elasticsearch `/el`

| Метод | Эндпоинт | Описание |
|---|---|---|
| `POST` | `/el/create_index` | Создание индекса Elasticsearch с произвольным телом запроса |
| `POST` | `/el/delete_index` | Удаление указанного индекса |
| `POST` | `/el/create_index_products` | Создание индекса товаров с ngram-анализатором (3–15 символов) для частичного поиска |
| `POST` | `/el/add_all_products` | Асинхронная полная переиндексация всех товаров из БД |
| `POST` | `/el/add_all_products_sync` | Синхронная полная переиндексация товаров (используется при первом запуске проекта) |

### ⚙️ Redis `/redis`

| Метод | Эндпоинт | Описание |
|---|---|---|
| `GET` | `/redis/test` | Тестовый эндпоинт для проверки работы кэширования (TTL 60 сек) |

### 🔧 Административная панель

SQLAdmin-панель, доступна по адресу `/secret-admin-path` (защищена middleware, проверяющим роль `admin`). Позволяет просматривать и редактировать: пользователей, заказы, товары, категории, отзывы, магазины, остатки товаров, историю поиска, избранное.
